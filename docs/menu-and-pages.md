# 菜单与页面规划

## 前台主菜单

固定菜单：

1. 首页：`/`
2. 动态：`/news`
3. 日报：`/daily`
4. 美容：`/beauty`
5. 品牌库：`/brands`
6. 声量观察：`/public-voice`
7. 样本库：`/samples`
8. 关于：`/about`

暂不展示：榜单、权威榜、排行榜、指数、白皮书、蓝皮书、报告商城、会员。

## 美容频道定位

美容频道是第一阶段重点频道，定位为：宠物美容、洗护、护理、造型、洗护用品、门店洗美、上门洗护、美容师与美容趋势的公开信息观察频道。

二级栏目：

1. 美容动态：`/beauty/news`
2. 洗护用品：`/beauty/products`
3. 门店洗美：`/beauty/stores`
4. 上门洗护：`/beauty/mobile-grooming`
5. 美容师：`/beauty/groomers`
6. 美容趋势：`/beauty/trends`

## 美容采集关键词

核心关键词：宠物美容、宠物洗护、宠物洗美、宠物SPA、宠物造型、宠物护理、宠物美容师、宠物洗护用品、宠物香波、宠物护毛素、宠物上门洗护、宠物到家洗护、移动宠物洗护、宠物洗护车、狗狗洗澡、猫咪洗澡、狗狗美容、猫咪护理、宠物门店洗护、宠物店洗美。

趋势关键词：成分党、Clean Beauty、拟人化护理、功能型洗护、高端洗护、情绪价值、宠物香氛、宠物皮肤护理、宠物敏感肌、宠物除臭、宠物口腔护理、宠物耳道清洁、宠物护理标准化、宠物美容职业化。

业务关键词：上门洗护、上门宠物美容、宠物移动洗护、宠物洗护预约、宠物洗护价格、宠物洗护套餐、宠物美容培训、宠物美容加盟、宠物美容门店、宠物美容连锁。
